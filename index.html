About Expense Tracker

The Expense Tracker helps you keep track of your daily expenses. It features an intuitive form for data entry and interactive visualizations to help you analyze your spending habits.

How to Use

Follow these steps to get started with the Expense Tracker:

  1. Download and extract the repository using the button above.
  2. Ensure you have Python installed on your system (version 3.7 or higher).
  3. Install the required dependencies by running pip install -r requirements.txt in the extracted folder.
  4. Start the application by running python app.py.
  5. Open your browser and navigate to http://127.0.0.1:5000/.
  6. Add your expenses, view visualizations, and track your spending habits!
